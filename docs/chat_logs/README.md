# チャット記録（chat_logs）

stock-alert オーナーと Claude のセッション内容を、**日付ごとに1ファイル**で残すフォルダ。

## 命名規則
- `YYYY-MM-DD.md`（例: `2026-06-09.md`）
- 1日に複数セッションがあれば同じ日付ファイルに追記する

## 中身
- その日に話したこと・決めたこと・やったことを**時系列・トピックごと**に記録
- 逐語の全文ではなく、後から振り返って分かる粒度の要約

## 運用（Claude向け）
- セッションの区切り／終了時、または「今日の記録を残して」と言われたら、その日の `YYYY-MM-DD.md` を作成・追記する
- 作成・追記は許可不要の定常作業として扱ってよい（CLAUDE.md 0章の「許可を取る」対象外。記録のみで本番に影響しないため）
- コミットは他の変更と混ぜず、`docs/chat_logs/` 単独で行う
