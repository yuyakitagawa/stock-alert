## qv 戦略の設計（2026-06-11追加）

**仮説:** 業績が強い（Piotroski高）× 株価が低迷している（52週安値圏）
= 市場の見落とし → 遅れた評価が株価に反映される（Value × Quality のリバウンド）

**買いゲート（全通過で採用・net降順）:**
- `Piotroski ≥ 0.67`（6/9以上: 財務健全）
- `pos52 < 0.45`（52週安値圏: 株価低迷）
- `bps_growth > 0 OR eps_surprise > 0`（業績改善シグナル）
- `drawdown60 > -15%`（フリーフォール除外）
- `dp ≤ 8`（急落リスク許容値）
- `vol20 ≤ 25`（過大ボラ除外）
- 相場ゲートなし（逆張りのため・暴落期も仕込む）

**売り:** drop_prob ≥ 10 / net < -5% / トレーリングストップ(1.5×ATR) / 90日上限

データが積まれるにつれ統計的信頼性が上がる。

## 主要な知見

1. **モデルの実力は下落回避（AUC 0.79）に偏る。上昇当ては弱い（AUC 0.66）。**
   → モメンタム戦略では自滅する（高い株をdrop_probで投げてしまう）。
   → Quality×Value フィルターがモデルの弱点を補うことで日経超えを達成。

2. **「業績強×株価低迷」フィルターがモデルの下落回避力と最もよく組み合わさる。**
   Piotroski高・pos52低の銘柄は「上昇ポテンシャルが大きく、下落余地は限定的」という特性。
   これがモデルの下落回避力と相乗効果を生み、DD最小かつ高リターンを実現。

3. **相場ゲートは「買い条件」に使うのが正解。「売りトリガー」にするとwhipsaw自滅。**
   qv は逆張りのため相場ゲートなし（暴落期に安値で仕込む）。

4. **逆張り（qv）は保有期間が長い方が機能する（大半が90日時間切れ退場）。**
   平均保有90日 → 市場が業績を「再認識」するまで待つ設計。
   モメンタム戦略（即時反応前提）とは真逆のアプローチ。

5. **集中5銘柄・少数トレードの解釈には注意が必要。**
   日本新薬のような超大勝ちが全体を押し上げる可能性がある。

## 現在の最良戦略: qv

- CAGR 46.1%（強気期間）/ bear +2.7%（暴落耐性）
- ドローダウン最大 -5.9% 強気 / -3.6% bear（全戦略最小）
- 日経225超え（+72.7% vs +60.7%）
- 1000万円 → 1億円: 約6.1年（CAGR継続前提）
