# stock-alert

東証上場株式の機械学習スクリーニング・アラートシステム。毎日自動でランキングを生成し、メールで通知する。

## システム概要

毎朝8時（JST）にGitHub Actionsで自動実行される。

    screener.py → rf_train_v3.py → rank_stocks.py → alert_email.py
       (毎日)       (土曜のみ)        (毎日)            (毎日)

---

## ファイル構成

| ファイル | 役割 |
|---|---|
| screener.py | JPX全銘柄から条件通過銘柄を抽出してCSV保存 |
| rf_train_v3.py | XGBoostモデルを東証全銘柄×5年データで学習 |
| rank_stocks.py | スクリーナー通過銘柄に上昇/下落確率をつけてランキング |
| alert_email.py | ランキング上位をGmailでメール送信 |
| generate_post.py | SNS投稿用テキストを生成 |
| sheets_helper.py | Googleスプレッドシートへの書き込みユーティリティ |

## モデル詳細（rf_train_v3.py）

### 学習データ
- 対象：東証プライム・スタンダード全銘柄（約3,500〜4,000銘柄）
- 期間：過去5年分（約1,800日）
- サンプリング：20営業日ごと（自己相関を低減）
- 分割：2025年1月1日より前→学習用 / 以降→テスト用（ウォークフォワード）

### 特徴量（75次元）

テクニカル10次元
- ret5, ret20, ret60, ret90：5/20/60/90日リターン
- ma5_25, ma25_75：移動平均クロス（5-25日、25-75日）
- rsi：RSI（14日）
- vol20, vol60：ボラティリティ（年率換算）
- pos52：52週レンジ内の位置

トレンド反転5次元
- drawdown60：直近60日高値からの下落率
- from_hi52：52週高値からの乖離率
- down_streak：連続下落日数（0〜1に正規化）
- momentum_accel：短期モメンタム加速度（ret5 - ret20/4）
- ma_cross_dir：MA5/MA25クロスの方向変化

リターン系列60次元
- 過去60日間の日次リターン系列（±20%にクリップ）

### 予測ラベル
- 上昇モデル：63営業日後（約3ヶ月）に+15%以上
- 下落モデル：63営業日後（約3ヶ月）に-15%以下

### モデル精度（v3学習結果）
- 上昇モデル AUC：約0.60
- 下落モデル AUC：約0.77

### 学習タイミング
- GitHub Actionsで毎週土曜に自動再学習
- early_stopping_rounds=30 で過学習を抑制

## ランキングロジック（rank_stocks.py）

ネットスコア = 上昇確率(%) − 下落確率(%)

### フィルター（除外条件）
- 連続下落日数が3日超（down_streak > 0.15）
- 直近60日高値から15%超の下落（drawdown60 < -0.15）

### 推奨ラベル

| 条件 | ラベル |
|---|---|
| ネット>=10 かつ ボラ低 | 買い可能性あり |
| ネット>=5 かつ ボラ低 | 買い可能性あり |
| ネット>=5 かつ ボラ高 | 買い可能性あり（荒れ注意）|
| ネット<-10 かつ ボラ低 | 売り可能性あり |
| ネット<-5 かつ ボラ低 | 売り可能性あり |
| ネット<-5 かつ ボラ高 | 売り様子見 |
| それ以外 | 様子見 |

### 日経225比較
各銘柄の5/20/60日リターンから日経225の同期間リターンを引いた相対強度も表示。

## 設計上の注意点

- モデルの限界：AUC 0.60はランダム（0.50）よりわずかに良い程度。参考指標として使い、最終判断は自分で行う。
- フィルターの意味：下落フィルターはルールベースの除外。モデルが「買い」と言っても直近で崩れている銘柄を弾く。
- ウォークフォワード分割の意義：将来データで学習する「先読みバイアス」を防ぐため、時系列を守って学習/テストを分割している。

## モデル改善履歴

### v4（2026-04-30）AUC 上昇0.644 / 下落0.780 ← 現在

**追加した特徴量（81次元）**
- 出来高3次元：vr520（5日/20日平均出来高比）、vr2060（20日/60日）、vsurge（直近/20日）
- 日経マクロ3次元：日経225の5/20/60日リターンを各サンプル時点で付加

**変更したこと**
- 出来高・日経マクロを入力特徴量として追加 → AUC改善
- 相対リターンラベル（日経比）を試みたが精度低下のため撤回
- 予測期間1ヶ月（21日）を試みたが精度低下のため3ヶ月（63日）に戻した
- early_stopping_rounds=30 をコンストラクタに追加（上昇88エポック、下落168エポックで停止）

### v3（2026-04-28）AUC 上昇0.603 / 下落0.767

- TSE全銘柄（プライム・スタンダード）× 5年データで学習
- ウォークフォワード分割導入（2025-01-01で学習/テスト分離）
- 20営業日ごとサンプリング（自己相関低減）
- 75次元特徴量（テクニカル10 + トレンド反転5 + 60日系列）

### v2以前

- スクリーナー通過銘柄のみで学習 → 先読みバイアスあり
- ランダム500銘柄サンプリング → サンプル数不足
